1. 프롬프트 엔지니어링 기초

1.1 프롬프트란?

프롬프트는 AI 모델에게 주어지는 입력 텍스트로, 원하는 결과를 얻기 위한 지시사항입니다.

1.2 좋은 프롬프트의 특징

  • 명확성: 구체적이고 명확한 지시
  • 맥락성: 충분한 배경 정보 제공
  • 구조화: 논리적인 순서와 구조
  • 목적성: 원하는 결과에 대한 명확한 설명

2. 기본 프롬프트 기법

2.1 Zero-shot 프롬프팅

예시 없이 직접적인 지시만으로 작업을 수행하도록 하는 기법

예시:
"다음 문장을 한국어로 번역해주세요: 'Hello, how are you?'"

2.2 Few-shot 프롬프팅

몇 개의 예시를 제공하여 패턴을 학습하도록 하는 기법

예시:
"긍정 → 부정으로 변환:
좋다 → 나쁘다
밝다 → 어둡다
빠르다 → ?"

3. 고급 프롬프트 기법

3.1 Chain-of-Thought (CoT)

단계별 사고 과정을 명시적으로 요청하는 기법

"문제를 단계별로 해결해주세요:
1. 문제 분석
2. 해결 방법 도출
3. 결과 검증"

3.2 Role-based 프롬프팅

AI에게 특정 역할을 부여하여 전문성을 높이는 기법

"당신은 경험 많은 데이터 과학자입니다. 다음 데이터를 분석해주세요..."
